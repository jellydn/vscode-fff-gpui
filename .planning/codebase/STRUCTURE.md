## Naming Conventions

- **Files**: camelCase (`findFiles.ts`, `openFiles.ts`, `client.test.ts`)
- **Functions**: camelCase (`sendCommand`, `resolveSocketPath`, `openFiles`)
- **Interfaces**: PascalCase (`ServiceCommand`, `PickEntry`, `PickResponse`)
- **Type guards**: `is` prefix (`isPickResponse`, `isPickEntry`)
- **Command IDs**: dot-separated (`fff-gpui.findFiles`, `fff-gpui.grepFiles`)
- **Config keys**: dot-separated (`fff-gpui.socketPath`)
- **Message prefix**: `fff-gpui:` in all user-facing messages

## Bundle Output

- `dist/extension.js` — single CJS bundle (tsup, external: vscode, noExternal: reactive-vscode)
- VSIX includes: `dist/`, `logo.png`, `README.md`, `package.json`, `LICENSE`
- VSIX excludes: `src/`, `test/`, `doc/`, `.github/`, config files (`.vscodeignore`)
